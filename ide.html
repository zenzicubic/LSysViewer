L-System Editor/IDE

This tool allows you to build L-systems, or Lindenmayer systems.

These are L-systems, or Lindenmayer systems. They were invented by Aristid Lindenmayer in 1968 to model the growth of plants. They use a formal grammar and string rewriting system to represent instructions which the computer follows to draw a picture. They start with an initial state or axiom, and apply rules to rewrite that string.

Default Commands:

  • +: Turn left θ degrees.
  • -: Turn right θ degrees.
  • F: Move forward n steps.
  • [: Push the current state onto the stack.
  • ]: Pop the most recent state from the stack and navigate to it.

Any number of arbitrary flow-control symbols may also be used.
